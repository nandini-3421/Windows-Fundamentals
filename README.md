# Windows Fundamentals for IT & Cybersecurity

This repository provides a detailed yet beginner-friendly explanation of the
Microsoft Windows operating system. It focuses on how Windows works internally,
how users interact with it, and how administrators troubleshoot and secure it.

## Who this repo is for
- CCNA learners
- Cybersecurity & SOC beginners
- IT Support & System Admin aspirants
- Freshers preparing for technical interviews

## Topics Covered
- Windows Architecture
- Users & Groups
- NTFS File System & Permissions
- Windows Networking Basics
- Windows Security Basics
- Important Windows Commands
- Troubleshooting
