# Windows Networking Basics

Windows uses the TCP/IP protocol suite for networking.

## Key Networking Components
- IP Address: Identifies device
- Subnet Mask: Defines network range
- Default Gateway: Route to other networks
- DNS Server: Resolves domain names

## Useful Networking Commands
ipconfig
ping
tracert
netstat
